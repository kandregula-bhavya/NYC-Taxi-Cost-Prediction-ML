# 🚖 Taxi Trip Cost Prediction using Machine Learning

## 📌 Project Overview
This project aims to predict the total cost of taxi trips using machine learning models. The dataset includes features such as passenger count, trip distance, and fare-related attributes.

---

## 🎯 Objective
To predict the total_amount of taxi trips using regression techniques and achieve high accuracy.

---

## ⚙️ Techniques Used
- Data Preprocessing
- Feature Extraction
- Data Visualization
- Machine Learning Models

---

## 🤖 Models Used
- Linear Regression
- Multiple Linear Regression
- Polynomial Regression

---

## 📊 Evaluation Metric
- R² Score

✅ Best Model: **Multiple Linear Regression**  
✅ R² Score: **0.9484550**

---

## 📂 Project Structure
- preprocessing.ipynb
- visualization.ipynb
- model.ipynb
- report.pdf

---

## 📈 Key Features
- Feature extraction from datetime columns
- Data cleaning and preprocessing
- Model comparison

---

## 🚀 Conclusion
Multiple Linear Regression performed best in predicting total taxi trip cost with high accuracy.

---

## 📎 Dataset
NYC Taxi Trip Dataset
